# Agent Skills

This folder contains localized skill definitions meant for the AI agent. Each markdown file describes how the agent should execute a specific type of task within this project:
- `audit_code.md`: Instructions for code review and auditing.
- `deploy_app.md`: Instructions for managing deployments.
- `generate_code.md`: Coding standards and generation instructions.
- `write_specs.md`: Instructions for writing technical specifications.
