# Project Root

This is the main project repository. It contains the structure needed for an AI-agent-driven development workflow.

> **Note**: This workflow is from the GDG workshop.

## Directory Structure

- `.agents/`: Contains configurations, specialized skills, and automated workflows for AI agents (e.g., Antigravity).
- `app_build/`: Designated folder for application build outputs, compiled assets, and deployment-ready binaries.
- `production_artifacts/`: Used for storing production-related artifacts, such as technical planning documents and finalized specifications.
